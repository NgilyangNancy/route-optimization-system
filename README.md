#### Route Optimization System



* Description:

Python project to find shortest distances and paths in a graph using Dijkstra's algorithm.

Supports both terminal and GUI input for nodes and edges.



* Files:

\- graph.py     : Graph structure

\- dijkstra.py  : Shortest path calculation

\- main.py      : Terminal version

\- gui\_main.py  : GUI version



* How to use:

1\. Terminal: python main.py

   - Enter edges (e.g., "A B 5"), type 'done', then starting node

   - Shows distances and paths



2\. GUI: python gui\_main.py

   - Enter edges and starting node in popup boxes

   - Results appear in a popup window
